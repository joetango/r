  1. We find a very favorable ROC plot with an area of 0.9974 under the curve, which indicates a good fit. The area under curve is useful because it can be used to measure the efficacy of different models.
predprob <- predict(subsetmodel, type = "response")

wbcam <- mutate(wbca, residuals = residuals(subsetmodel), linpred = predict(subsetmodel))

mod_roc = roc(wbca$Class, predprob)
## Setting levels: control = 0, case = 1
## Setting direction: controls < cases
plot(mod_roc)

auc(mod_roc)
